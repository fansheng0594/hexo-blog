MacOS 系统安装 NodeJS

​ 在日常开发中，项目需要依赖 NodeJs 运行环境。但由于不同项目可能会依赖不同的 NodeJs 版本，这时就需要一个 NodeJS 的版本管理工具。

​ 这里我们选择 NVM 版本管理工具，来解决一台机器上多个项目 NodeJS 版本不兼容的问题。

  1. 安装命令行工具

如果之前已经安装过，则直接忽略该步骤。

通过命令行安装：

xcode-select --install

  1. 安装 NVM

命令行安装：

curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh | bash

配置环境变量，将以下代码加入到启动脚本 ~/.zshrc 或 ./bash_profile，重启命令行会话或 source 脚本后，即可使用。

export NVM_DIR="$HOME/.nvm" [ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh" # This loads nvm

  1. 测试

查看 NVM 版本测试是否安装成功

nvm --version

PHP的三个主要领域：

  • 服务端脚本。需要PHP解析器、web服务器、web浏览器
  • 命令行脚本。仅需要PHP解析器来执行
  • 桌面应用程序。PHP-GTK

PHP支持绝大多数的web服务器，数据库，系统

超全局变量 $_SERVER

Post 并不会比 get 更安全。两者之间的选择原因有多种，一是 url 往往有长度限制，不能发送非常多请求，二是，谷歌搜索这一类在url中传递搜索条件，利于搜索者们进行方便分享。用post传输只是比get传输泄漏信息较难些罢了，如果在 get 中传输密码，很容易被他人查看，或用户不慎分享了 url 就会造成损失。然而，如果没有部署 ssl 服务，两者都是不安全的

对于 Apache，PHP有一个直接模块接口。如果 PHP 不能作为模块支持 web 服务器，可以将其作为 FastCGI 处理器来使用，这就意味着 CGI 可执行程序来处理服务器上的所有 PHP 文件请求

纯PHP脚本不加 ?> 结尾标记的原因是，PHP输出时会自动删除结束符后的第一个换行，如果结束符后有5个换行，则会输出4个空行

结束标记在单行注释中会被解析（/ #），在多行注释（/**/）中不会

自控力课程笔记

这本书的内容是十周的课程，非常需要花十周的时间来一步一步地实施策略。花一整周的时间观察每章内容所传授的想法在实际生活中的应用情况。不管我是花多少时间看完这本书，我都希望自己能花十周时间，每周都整理本书的想法对我的影响，尽可能地去应用这些想法。

第一周

自知之明是自控力的基础。自认为意志坚定的人反而最容易在诱惑面前失控，因为他对自己还不够了解，他们无法预测自己在何时何地，会由于什么原因失控。

意志力的三部分：“我要做”，“我不要”，“我想要”。

  • 我要做。对于打算拖延的事情，我们要说，“我要做”
  • 我不要。在诱惑面前，我们要学会，“我不要”
  • 我想要。要想在需要“说不”时“说不”，在需要“说好”时“说好”，则需要牢记自己真正想要的是什么，这就是“我想要”

人脑很早就学会了自控，如部落之间互相帮助，为了自己生病时无法打猎而饿死。不去偷东西，避免被赶出部落。

前额皮质氛围三个区域，分管着”我不要“，”我要做”，“我想要”。

在诱惑面前，两个自我发生分歧的时候，总会有一方击败另一方。决定放弃的一方并没有错，只是双方觉得重要的东西不同而已

我把生气愤怒的自己称为“狂暴巨兽”，爱抱怨和爱嘲讽的自己称为“键盘侠”，把总是拖延的自己称为“拖延者”。在他们占据上风的时候，我要意识到他们的存在，并唤醒理性的我，唤醒意志力

或许有人觉得欲望和恐惧对自控力来说是个负能量，实际上，欲望和恐惧是至关重要的。如果没有了欲望，自己将会很沮丧；如果没有了恐惧，将无法保护自己受到伤害。所以我们要利用这两个本能而不是对抗它们。

自我意识。大多数人在做决定的时候，根本意识不到自己为什么作决定。所以，注意力分散的人更容易向诱惑屈服，商店里的促销活动会让购物者买一些毫无价值的商品。

要想有更强大的自控力，就需要有更多的自我意识。把你作的决定都记下来，在一天结束的时候，回顾这一天所做的决定，哪些有利于实现目标，哪些会消磨你的意志。坚持作记录，有助于减少在注意力分散时作决定，每天坚持下去，同时也增强了意志力

观察一下，自己是怎么屈服于冲动的，只需要看看你能否及早意识到自己在做什么，什么样的想法，感受和情况会让你更容易冲动。而这时候，想些什么，做些什么又能让你放弃冲动

人脑像一个求知欲很强的学生，对经验有着超乎想象的反应。如果你每天都学习数学，那你就更擅长数学；如果你每天都让它专注，它就会越来越专注；如果你每天都让它忧虑，它就会越来越忧虑

可以在家中布满陷阱，来挑战“我不要”

冥想。冥想可以提升你集中注意力，管理压力，克制冲动和认识自我的能力。坐在椅子上，背挺直，双手放在膝盖上，闭上眼睛，吸气时脑海中默念“吸”，呼气时脑海中默念“呼”。当你发现走神时，重新将注意力集中到呼吸上。几分钟后，你就可以不再默念“呼“和”吸“了。试着专注于呼吸本身。注意到空气从嘴巴和鼻子进入和呼出的感觉，感觉到胸腹部的扩张和收缩。不再默念之后，你可能会更容易走神，当你发现自己在想其他事情的时候，重新将注意力集中到呼吸上。实在很难集中的话就继续默念。这部分的训练能锻炼你的自我意识和自控能力。刚开始每天五分钟，习惯之后，每天锻炼十到十五分钟

更难的事是什么？想象你正在面临一个意志力挑战。更难的事是什么？为何它如此困难？

认清两个自我。你在做意志力挑战时，如何描述两个互相抗衡的自我？冲动的你想要什么？理智的你想要什么？

第二周

当你遇到真正的意志力挑战时，你的身体一定能感觉到，这是你身体内部的战斗。有时，欲望会占据上风。有时，那个更加明智、想变得更好的你会占据上风。只有大脑和身体同时作用，才有力量克服冲动

我们行动和努力的指南不可以是想象中的图像，应该是考虑清楚的概念。直观的印象和图像强有力地入侵我们的情绪，扰乱其平和或动摇其决心。这是因为眼前存在的、可以直观参照的事物总是以全力突然产生效果，轻易就让我们对其一目了然。然而，思想和理据却更需要平静和时间。但此时，我们无法随时把思想和理据呈现在眼前。正因为这样，经过思考我们已经放弃了某一令人垂诞的事物，但一看见这可欲之物，我们又被其吸引

如果我们无法通过思想来消除印象对我们的影响，最好的办法就是用相反的印象来中和它的作用。比如想到诱人的奶酪，我们可以想象一下，全身都是油脂，高高凸起的肚子，以及很不健康的你。

我们总觉得诱惑和麻烦来自外部的世界，是甜甜圈，奶酪或是发达的互联网。但其实问题出在我们身上，是我们的思想、欲望、情绪和冲动出了问题。对你的意志力挑战来说，最重要的是认清什么是需要克制的内在冲动。哪些想法或感觉使你在不情愿的时候作出决定？下一次受到诱惑的时候，试着观察一下自己的内心世界。

三思而后行，放慢速度

心率变异度反应了意志力的程度，心率变异度越高，意志力越强，反之。任何给你的身心带来压力的东西都会影响自控力的生理基础，甚至会摧毁你的意志力。这方面的敌人诸如焦虑、愤怒、抑郁和孤独等，慢性疼痛和慢性疾病也会消耗大脑的大部分意志力储备。

提高心率变异度可以通过冥想，锻炼，保持良好的睡眠，保证健康饮食等。还有一种较容易实现的方法是将呼吸频率降低到每分钟4-6次，也就是每次呼吸用10-15秒时间，比平常呼吸要慢

锻炼是提升意志力非常有效的方法，对起步者来说，锻炼能起到立竿见影的效果。不仅能缓解日常压力，还会抵抗抑郁，让你的头脑更充实，运转更快。任何让你离开椅子的运动，都能提高你的意志力储备。每次锻炼的时候，我们可以把锻炼当成是意志力发动机，我们希望每天变得更强，唯有把锻炼坚持下去，让自己有能力把控生活。

睡眠对意志力有很大的影响。长期睡眠不足会让你的意志力逐渐消耗殆尽，你会很难控制情绪，集中注意力。所以，为了获得更强的意志力，那就早点休息吧。

我们越晚入睡，说明我们做了更多的事情。此时，可以把“获得更多的睡眠”定义为“我不要”的意志力挑战。真正的问题并不是强迫自己去睡觉，而是远离那些让自己没法睡觉的事。

适当的自控力是我们健康生活中必不可少的一部分，但如果过度自控则会产生很大的压力，威胁到我们的身心健康。因为，每次的自控都需要消耗我们的体力，而我们还需要把体力花费在更多事情上。所以我们需要放弃对意志力的完美控制

意志力会受到多方面的影响，比如睡眠不足、饮食不良、缺少运动或是压力过大等。态度很重要，身体也不能掉链子。如果我们想更好地应对挑战，就需要更有效地管理压力、照顾自己。睡眠不足等坏习惯不仅反映出我们缺乏自控力，还消耗了体力，带来更多的压力，偷走了我们的自控力

试着找出持续一整天或一整周的压力，看看它对你的自控力产生怎样的影响。你有过强烈的欲望吗？你发脾气了吗？你把事情拖到明天做了吗？

意志力实验：

  • 呼吸出自控力。把呼吸调整到每分钟4-6次
  • 五分钟给意志力加油。出门活动，哪怕只是在周围转转
  • 保证足够的睡眠

第三周

我们只有一定量的意志力，一旦你将它消耗殆尽，你在诱惑面前就会毫无防备力，至少会处于下风。自控力就像肌肉一样有极限，使用之后就会疲惫不堪，如果不让肌肉休息，就会完全失去力量

不要浪费意志力最强的时间段，把意志力用在刀刃上，如果你觉得没有时间和精力去处理“我想要”做的事，那就把它安排在你意志力最强的时候做

血糖降低时，意志力也就跟着降低。

大脑是个小气鬼，在某个特定时刻，只会提供一小部分能量，而保存着大部分的能量。资源丰富的时候就会支出能量，资源缺少时就会保存能量。所以，最好在有能量时做选择和决定

突然增加糖分会在短期内提升意志力，但从长远的角度来看，过度依赖糖分不是自控的好方法。科学家推荐低血糖饮食，这样能让血糖更稳定，低血糖食品包括瘦肉蛋白，坚果和豆类，粗纤维谷类和麦片，大多数的水果和蔬菜。每天都吃一顿丰盛的早餐，吃零食时选择坚果而不是糖果

锻炼身体可能会让你的自控力肌肉感到疲惫，但经过一段时间的锻炼，它肯定能变得更强健

在一些小事上持续自控会提高整体的自控力，这些训练时为了养成习惯，通过看似不重要的意志力挑战来训练“意志力肌肉”，同时不用担心自己无法坚持到底

锻炼模式

  • 增强“我不要”的力量：不嘲讽，不挖鼻孔等
  • 增强“我想要”的力量：每天冥想，每天做卫生等
  • 增强自我监控能力：认真记录支出、饮食，或是记录上网、看书、学习的时间

如果你想彻底改变旧习惯，最好先找一种简单的方法来训练自控力、提高意志力，而不是设定一个过高的目标

运动疲劳的原因或许不是肌肉无法继续工作，而是大脑中过度保护性的监控机制发挥了作用，当运动员第一次想放弃的时候，其实他们远远没有达到自己的极限

疲惫不是一种身体反应，而是一种感觉，一种反应。疲惫只不过是大脑产生的某种反应，好让我们停下来。知道我们的意志力比想象中多得多，这确实是令人开心的事情。下一次你觉得疲惫的时候，觉得自己没法自控时，想办法挑战一下自己

当你的意志力告急时，你可以挖掘你的”我想要“的力量，让自己恢复能量。可以考虑以下动机：

  • 如果你挑战成功，你会收获什么？你个人会有什么回报？你会更健康、更幸福、更自由、更有钱，还是会更成功？
  • 如果挑战成功，还有谁会受益？肯定有人依赖于你，你的选择会影响到他们，你的成功会怎样帮到他们？
  • 如果你现在愿意做困难的事，那么一段时间后，这个挑战将会变容易。如果你知道还有更大的进步空间，现在的不适是不是变得更值得了？

如果我们想增强自控力，就要考虑如何支撑住最疲惫的自己，而不是指望最理想的自己突然出现来拯救生活

我们不能控制所有的事，但提高自控力的唯一办法就是提升我们的极限

记录意志力的上下波动。注意什么时候意志力最强，什么时候最容易放弃

意志力实验：

  • 意志力饮食。确保身体摄入了足够的食物，能为你提供更多的能量
  • 意志力锻炼。选择一件事来做或不做，记录一件不曾关心的事
  • 发现你的“我想要”的力量，每当要放弃的时候，想一想它

裸辞两个月后的回忆

裸辞已经有两个月了，来说说我这两个月的一些生活感受和总结吧。

辞职是因为公司爆发了，开始了各种的裁员，也迟迟发不出工资，觉得公司迟早要倒闭，趁早脱坑换一个环境重新开始吧。原先在离职前就开始找工作了，也已经面试了两家了，我想，接下来的日子里可以专心把精力投入到找工作中，能够快速找到一份满意的工作。之前也担心说自己辞职后无法在家里认真学习，就物色了一家专业自习室，让自己每天的学习都在那里进行，面试完后就过去做总结学习。然而，就在真正离职后的两天里我就突然改变了想法了，我希望利用一个月的时间好好来进行学习，学习《数据结构与算法》、《深入理解计算机系统》、《Java核心思想》等。由于之前约了一根面试，想着把这个面试面完吧，而那一家考了很多基本功，自己在这次面试碰了壁，就更加坚定了我学习计算机基础的心，于是我决定花一个月的时间来进行学习知识，不再去想面试的事情，专心学习接下来的课程。

想法是美好的，现实是残酷的，很快，我在学习算法中就遇到了难题，这本书是用 C 语言来进行编写的，很快我就放弃了，选择了《算法导论》，这个计划后来很快也宣告了停止，原因是难度太大，自己畏缩了吧。《深入理解计算机系统》卡在了汇编语言上了，后面越学越糊涂，于是开启了《汇编语言》的学习，也是看着B站的教学视频跟着学习。《Java核心思想》是学得最多的吧，也算是复习了一下基础，也看到了Java的丰富性。在办了两次周卡后，决定开始办月卡了，回想起来，这笔钱亏了好多，自己估计只去了半个月左右不到吧。月卡办完两个礼拜后，自己就每天各种睡懒觉，各种不想学习，意志消沉，开始逐渐不去自习室了。我给自己找了很多理由：来回要花很多时间、座椅不舒服、不能很好睡一个午觉、在家能更自由一点学英语、在那不自在等等。就这样，接下来十几天几乎就再也没去过自习室了，前面的基本技能书基本也再也没有打开过了。本来之前每天也有去工大跑步的，但是后来由于军训关闭了，虽然八月八号就开了，但是我看成十八号才开，那一阵子也停止锻炼了，有很多天都没出门，偶尔下楼倒下垃圾和取快递，接近废人了。

没去自习室的这段时间又开始决定在思想上和英语上有提升。在这段时间里开始了英语学习和思维学习，这段时间里买了哈佛毕业生的一个学习力课程，她从中推荐了几本书，看了本成长型思维的书，后来又看了本自控力的书，英语也学习了《美语入门》十段对话。

很快，就到了约定期八月十五号，这是我之前决定停止学习的日子。然而，我又怎么能停下来呢，压力越大越无力，基本上都到今天的十来天更混了，一拖再拖。直到今天。

今天我躺在床上睡觉的时候我想起来，我这么做不就是逃避压力嘛，我看完了《自控力》后，跟别人讲了一大堆道理，为什么自己没有觉悟呢？我恍然大悟，我起床，我告诉自己，我要勇敢面对压力，逃避压力对我来说是非常错误的决定，我要用挑战反应替代恐惧反应，结束这个糟糕的生活。我今天已经感到有点孤单无助了，感觉这样下去整个人就荒废了，而且经济开始紧张了，压力开始大起来了。我告诉我自己，拼尽全力结束它，开始新的生活吧，不要再这么每天毫无意义地过下去了。找工作是很痛苦，也很难，但是不踏出这一步，你将继续深深地陷入在沼泽中，无法自拔，直到精神消亡。

接下来总结一下我这两个月得到了哪些好处，又有哪些坏处，以及这段经历给我带来了哪些经验。

好处

  • 我看到了我在计算机基础这方面是非常薄弱的，觉得自己学习这些计算机基础也是没错的，只有掌握了基础功才不会盲人摸象。既然开始了一段学习，以后就继续学习下去
  • 在自习室看到了很多人在雅思、托福以及考验英语上下苦工，我也认为我可以做到，我也在这两个月里有了出国留学梦，自己一定要出国去世界看看，英语这门技能是接下来必须要掌握的一项核心技能，在工作上或是出国都用得上
  • 看了《看见成长的自己》和《自控力——和压力做朋友》这两本书，一个是讲述成长型思维和创新型思维，一个是告诉我们去拥抱压力，对我来说影响是挺大的，受益终身的。也告诉我自己以后要多看这类有关于思维模式的书
  • 自己松散了两个月，这两个月是一个很好的忠告，亲身体验到，生活越忙越有意义

坏处

  • 身体垮了，自控力太差，不爱惜身体
  • 没赚到钱
  • 浪费了很多时间学习，也许边上班边学习学得会比现在要多得多
  • 在家独自一人经常会易怒
  • 与人交流的机会少了

经验

  • 专注。有很多事情要做，有很多内容要学习，有很多书要看。但是，要有始有终，最好的办法就是，读完一本再读另一本，不然会导致很多事情做到一半半途而废，颗粒无收，而且心情也不好了。凡事要有优先级或者个排序，按照顺序来
  • 下午少睡，睡二十分钟到半个小时就足矣，否则头脑也会很累
  • 每天要有一个计划，给自己定一个不高也不低的目标，因为往往自己达到目标线后就再无动力继续往前了。还要做到日事日毕
  • 照顾好自己的身体，你现在的努力就是为了将来更好的生活，如果你不爱惜你自己的身体，你一定会过得很惨的
  • 拥抱压力吧
  • 用成长型思维处事
  • 每天越忙你才会觉得越有意义
  • 输入+输出型学习模式，多余人分享自己所学到的东西
  • 多训练“心流”状态
  • 多反思生活中的机会成本和沉没成本
  • 多倾听、观察、思考
  • 毅力很关键，找到自己每天的学习动力
  • 为团队做贡献，为更宏观的目标而去努力
